Random Movie Roulette is a popular Letterboxd list with which a user can use RANDOM.org to randomly choose a movie from a list of thousands.
This is a fun and excting way to choose films but scrolling through list pages can be tedious, and using unnumbered lists can make this method tough. With Random Movie Roulette, randomising letterboxd lists can be done easily.
